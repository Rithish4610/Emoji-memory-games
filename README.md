# Emoji Memory Game

A **fun and interactive memory card game** built using **HTML, CSS, and JavaScript**. Players test their memory by matching emoji pairs in the least moves and shortest time possible. This project is **fully front-end**, runs on any browser, and is perfect for showcasing **JavaScript logic, DOM manipulation, animations, and UI skills**.

## Features

- **Multiple Difficulty Levels:** Choose Easy (4x4) or Hard (6x6) before starting the game.
- **Animated Star Rating:** Performance is rated with 1-3 stars based on moves and time.
- **Moves Counter & Timer:** Tracks the number of moves and time taken.
- **Emoji-Based Cards:** No external images needed; fun and visually appealing.
- **Responsive Design:** Works on desktop and mobile screens.
- **Restart Game:** Restart anytime while keeping the selected difficulty.
- **Interactive Animations:** Cards flip smoothly; stars animate after winning.

## How to Play

1. Open the game in a browser (Live Server recommended in VS Code).
2. **Step 1:** Select your difficulty (Easy or Hard).  
3. **Step 2:** Click "Start Game" to proceed to the game board.  
4. **Step 3:** Flip two cards at a time to find matching emoji pairs.  
5. Try to complete the game with **as few moves and time as possible** to get the maximum stars.  
6. Restart the game anytime using the "Restart Game" button.

## Tech Stack

- **HTML** for structure
- **CSS** for styling and animations
- **JavaScript** for game logic and DOM manipulation
